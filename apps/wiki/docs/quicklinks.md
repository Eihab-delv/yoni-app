### Installation Issues

1. Clear caches: `pnpm store prune`
2. Remove node_modules: `rm -rf node_modules pnpm-lock.yaml`
3. Reinstall: `pnpm install`

### Build Issues

1. Clean builds: `pnpm clean`
2. Rebuild packages: `pnpm build`
3. Check TypeScript: `pnpm compile`

### Mobile Issues

1. Clear Metro cache: `pnpm expo start --clear`
2. Reset project: `pnpm clean`
